# medical-clinical-analytics
診療データを「知恵」に変え、医療の質と安全を向上させる分析支援システム。

## 📈 システムの役割
- **医療安全（インシデント）分析:** ヒヤリハット事例と診療データの相関による事故防止。
- **DPC指標の可視化:** 疾患別入院日数や薬剤コストの分析によるクリニカルパスの最適化。
- **経営意思決定支援:** 病院の稼働率予測や、地域医療需要に基づく経営戦略の策定支援。

## 🛠 使用フレームワーク・技術
- **Language:** Python 3.x (Pandas, Scikit-learn)
- **Data Visualization:** Streamlit または Plotly
- **Infrastructure:** Jupyter Notebook / PyCharm (プロフェッショナルなデータ解析)
